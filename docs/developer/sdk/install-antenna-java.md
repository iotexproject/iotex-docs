# Install Antenna-Java

## Install By Maven

```xml
<dependency>
  <groupId>com.github.iotexproject</groupId>
  <artifactId>iotex-antenna-java</artifactId>
  <version>0.5.8</version>
</dependency>
```

## Install by Gradle

```java
implementation 'com.github.iotexproject:iotex-antenna-java:0.5.8'
```
